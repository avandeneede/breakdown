# repartition

Petite app HTML pour répartir un montant selon une clé de répartition configurable.

**Live:** https://avandeneede.github.io/repartition/

## Fonctionnalités

- Profils multiples (chacun avec ses propres clés)
- Mode pourcentage (%) ou parts pondérées
- Montant + devise (€, $, £, CHF)
- Égaliser / Normaliser à 100 %
- Export / Import JSON
- 100 % local — données stockées dans `localStorage`, aucun serveur
- Mobile-first — s'ajoute à l'écran d'accueil iOS/Android

## Utilisation

Ouvrez `index.html` dans un navigateur, ou visitez l'URL GitHub Pages ci-dessus.

### Sur iPhone
Safari → Partager → « Sur l'écran d'accueil » pour l'utiliser comme une app native.

## Activer GitHub Pages

Settings → Pages → Source: `main` branch, `/` (root) → Save.
